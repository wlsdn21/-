# 개인정보 처리방침

최종 업데이트: 2026-03-12

## 1. 개요

원화표시기(이하 "확장 프로그램")는 웹페이지에 표시된 외화 금액을 원화 기준으로 보기 쉽게 변환해 주는 Chrome 확장 프로그램입니다.

이 확장 프로그램은 기능 제공에 필요한 최소한의 정보만 브라우저 내부에서 처리하며, 별도의 회원가입 기능이나 자체 사용자 계정 시스템을 운영하지 않습니다.

## 2. 수집하거나 저장하는 정보

확장 프로그램은 아래 정보를 사용자 브라우저의 로컬 저장소에 저장할 수 있습니다.

- 환율 표시 on/off 설정
- 환율 소스 설정
- 사용자 지정 환율 값
- 억/조 단위 표시 여부
- 소수점 자리수 설정
- 사이트별 적용 목록
- 전체 사이트 적용 여부
- 다중 통화 활성화 설정
- 경제 지표 필터 설정
- 경제 지표 알림 설정
- 최근 환율 및 최근 경제 지표 캐시

위 정보는 확장 프로그램 동작을 위해 사용자의 브라우저 내부 저장소에 보관되며, 개발자 서버로 전송되지 않습니다.

## 3. 웹페이지 데이터 처리 방식

확장 프로그램은 사용자가 열어본 웹페이지 안에서 `USD`, `USDT`, `USDC` 등 통화 관련 텍스트를 감지해 KRW 값을 함께 표시합니다.

이 과정은 사용자의 브라우저 안에서 수행되며, 감지한 웹페이지 내용이나 변환 결과를 개발자 서버로 업로드하지 않습니다.

## 4. 외부 서비스 호출

확장 프로그램은 환율 및 경제 지표 기능을 위해 아래 외부 서비스에 요청을 보낼 수 있습니다.

- `open.er-api.com`
- `api.exchangerate-api.com`
- `api.exchangerate.host`
- `api.upbit.com`
- `api.manana.kr`
- `www.myfxbook.com`

이 요청은 최신 환율, 업비트 시세, 경제 지표 데이터를 가져오기 위한 목적에 한해 사용됩니다.

## 5. 수집하지 않는 정보

확장 프로그램은 다음 정보를 수집하거나 판매하지 않습니다.

- 이름, 이메일, 전화번호 등 직접 식별 가능한 개인정보
- 비밀번호, 결제 정보, 신용카드 정보
- 사용자의 계정 로그인 정보
- 개발자 서버로 전송되는 브라우징 기록 원문
- 광고용 추적 정보

또한 이 확장 프로그램에는 별도의 분석 SDK, 광고 SDK, 사용자 행동 추적 SDK가 포함되어 있지 않습니다.

## 6. 권한 사용 목적

### `storage`

사용자 설정과 캐시 데이터를 브라우저에 저장하기 위해 사용합니다.

### `alarms`

환율 및 경제 지표를 주기적으로 새로고침하기 위해 사용합니다.

### `notifications`

High Impact 경제 지표 발표 알림을 표시하기 위해 사용합니다.

### 웹사이트 접근 권한

현재 방문 중인 페이지 안의 통화 텍스트를 찾아 KRW 표시를 덧붙이기 위해 사용합니다.

## 7. 데이터 보관 기간

확장 프로그램이 저장하는 설정과 캐시 데이터는 사용자가 확장 프로그램을 삭제하거나 브라우저 저장소를 정리할 때까지 브라우저 내부에 남아 있을 수 있습니다.

## 8. 제3자 제공

확장 프로그램은 사용자의 개인정보를 개발자 서버나 제3자에게 판매하거나 임의로 제공하지 않습니다.

단, 환율 및 경제 지표 데이터를 가져오기 위해 외부 API 또는 RSS 서비스에 네트워크 요청이 발생할 수 있습니다.
