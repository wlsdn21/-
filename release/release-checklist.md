# 출시 체크리스트

## 1. 제출 파일 준비

- [ ] `release/chrome-web-store-listing-ko.md` 기준으로 스토어 이름과 설명문 확정
- [ ] `release/privacy-policy.md`를 외부 공개 URL로 게시
- [ ] `icon128.png` 최종본 확인
- [ ] 스크린샷 3~5장 캡처 완료
- [ ] `release/build-release-zip.ps1`로 배포 ZIP 생성
- [ ] 생성된 ZIP을 실제 Chrome 확장 프로그램으로 로드해 마지막 점검

## 2. 기능 점검

- [ ] USD, USDT, USDC가 정상적으로 KRW로 표시되는지 확인
- [ ] 국제 환율, 업비트 USDT, 업비트 USDC, 사용자 지정 환율 전환 확인
- [ ] 사이트별 적용과 전체 사이트 적용이 정상 동작하는지 확인
- [ ] EUR, JPY, CNY 토글이 정상 동작하는지 확인
- [ ] 경제 지표 목록 로딩과 필터 동작 확인
- [ ] High Impact 알림 설정 on/off 동작 확인
- [ ] 팝업에서 마지막 업데이트와 상태 표시 확인

## 3. 웹스토어 등록 폼 입력

- [ ] 제품 이름 입력
- [ ] 짧은 설명 입력
- [ ] 상세 설명 입력
- [ ] 카테고리 선택
- [ ] 언어 설정 확인
- [ ] 스크린샷 업로드
- [ ] 아이콘 업로드
- [ ] 개인정보처리방침 URL 입력

## 4. 권한 설명 준비

- [ ] `storage`: 사용자 설정과 환율 캐시 저장
- [ ] `alarms`: 주기적 환율 및 경제 지표 갱신
- [ ] `notifications`: High Impact 알림 표시
- [ ] 웹사이트 접근 권한: 페이지 내 통화 텍스트를 KRW로 표시하기 위해 필요
- [ ] 외부 네트워크 요청: 환율과 경제 지표 데이터 조회 목적

## 5. 출시 직전 권장 점검

- [ ] ZIP 안에 테스트 파일이 제외되어 있는지 확인
- [ ] 버전 번호가 manifest와 일치하는지 확인
- [ ] 팝업 내 깨진 문자열이나 레이아웃 이슈가 없는지 확인
- [ ] 지원 링크가 열리는지 확인
- [ ] 설명문에 실제 없는 기능이 적혀 있지 않은지 확인

## 6. 출시 직후 48시간 체크

- [ ] 웹스토어 심사 상태 확인
- [ ] 사용자 리뷰 또는 문의 채널 모니터링
- [ ] 환율 API 오류 여부 확인
- [ ] 자주 쓰는 사이트에서 변환 오탐/미탐 제보 수집
- [ ] 다음 버전 우선순위 정리

## 7. 우선순위 높은 후속 작업

- [ ] 지원 사이트 예시를 스토어 설명에 더 구체화
- [ ] 오탐/미탐 사례 수집
- [ ] 기본 소개 이미지를 더 깔끔하게 제작
- [ ] 웹스토어용 영문 설명도 별도 준비
