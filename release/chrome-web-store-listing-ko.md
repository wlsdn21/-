# Chrome Web Store 등록 문안

## 추천 스토어 이름

`원화표시기 | USD·USDT·USDC를 KRW로 바로 보기`

대안:

- `원화표시기 - 달러 가격을 원화로`
- `KRW Master Pro - USD/USDT/USDC to KRW`

## 짧은 설명

`웹페이지의 USD, USDT, USDC 가격을 실시간 환율 기준으로 KRW로 바꿔 보여주는 크롬 확장 프로그램`

## 한 줄 소개

`해외 가격을 볼 때마다 계산하지 말고, 바로 원화로 확인하세요.`

## 상세 설명

해외 거래소, 글로벌 투자 사이트, 경제 지표 페이지를 볼 때 달러 가격을 다시 계산하는 번거로움을 줄여주는 크롬 확장 프로그램입니다.

원화표시기는 웹페이지 안의 `USD`, `USDT`, `USDC` 금액을 감지해 한국 원화 기준으로 자연스럽게 함께 표시합니다. 필요하면 `EUR`, `JPY`, `CNY`도 추가로 켤 수 있고, 사이트별 적용과 경제지표 알림까지 한 번에 관리할 수 있습니다.

### 이런 분께 추천합니다

- 해외 거래소에서 달러 기준 가격을 자주 보는 사용자
- 크립토 시세를 USDT, USDC 기준으로 확인하는 사용자
- 경제 지표 페이지를 자주 확인하는 트레이더
- 웹에서 보는 숫자를 빠르게 원화 기준으로 이해하고 싶은 사용자

### 주요 기능

- 웹페이지의 `USD`, `USDT`, `USDC` 금액을 자동으로 `KRW`로 표시
- `업비트 USDT`, `업비트 USDC`, `국제 환율`, `사용자 지정 환율` 중 기준 선택
- 억/조 단위 표시와 소수점 자리수 설정
- 사이트별 적용 또는 전체 사이트 적용 모드
- `EUR`, `JPY`, `CNY` 추가 변환 지원
- 경제 지표 일정 보기와 High Impact 알림
- 팝업에서 환율 상태, 마지막 업데이트, 자동 갱신 상태 확인

### 왜 유용한가요

- 해외 가격을 볼 때마다 계산기를 열 필요가 없습니다.
- 거래소마다 다른 기준 환율을 선택할 수 있어 더 실전적인 판단이 가능합니다.
- 자주 쓰는 사이트만 골라 적용할 수 있어 화면 오염을 줄일 수 있습니다.

### 권한 사용 이유

- `모든 웹사이트에서 데이터 읽기`
  - 현재 열어본 페이지 안의 통화 텍스트를 찾아 KRW 표시를 붙이기 위해 사용합니다.
- `storage`
  - 환율 소스, 사이트 목록, 표시 옵션, 알림 설정을 브라우저에 저장하기 위해 사용합니다.
- `alarms`
  - 환율과 경제 지표를 주기적으로 새로고침하기 위해 사용합니다.
- `notifications`
  - High Impact 경제 지표 발표 알림을 표시하기 위해 사용합니다.

### 외부 데이터 소스

- 일반 환율 API
- Upbit 시세 API
- 경제 지표 RSS 피드

이 확장 프로그램은 별도 회원가입 없이 동작하며, 사용자의 계정 정보나 결제 정보를 수집하지 않습니다.
